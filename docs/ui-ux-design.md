# UI/UX Design - EcoMate

## Design System

### Color Palette

**Primary (Eco Green):**
- Primary 600: `#059669` (emerald-600)
- Primary 500: `#10b981` (emerald-500)
- Primary 400: `#34d399` (emerald-400)

**Neutral:**
- Gray 900: `#111827` (text primary)
- Gray 600: `#4b5563` (text secondary)
- Gray 200: `#e5e7eb` (borders)
- Gray 50: `#f9fafb` (background)

**Semantic:**
- Success: `#10b981` (green)
- Warning: `#f59e0b` (amber)
- Error: `#ef4444` (red)
- Info: `#3b82f6` (blue)

### Typography

- **Font Family**: Inter (Google Fonts)
- **Headings**:
  - H1: 2.5rem (40px), font-weight: 700
  - H2: 2rem (32px), font-weight: 600
  - H3: 1.5rem (24px), font-weight: 600
- **Body**: 1rem (16px), font-weight: 400
- **Small**: 0.875rem (14px)

### Spacing

Using 8px grid system:
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

## Responsive Design

### Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Mobile Adaptations
- Hamburger menu for navigation
- Stacked layouts (no sidebars)
- Bottom tab bar (Home, Search, Sell, Messages, Profile)
- Collapsible filters
- Touch-optimized buttons (min 44x44px)
- Swipeable image galleries

- Keyboard navigation support
- High contrast mode support
- Screen reader optimized
- Focus indicators
- Alt text for all images
