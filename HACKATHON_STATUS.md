## 技術スタック

### バックエンド
- **言語**: Go 1.21+
- **フレームワーク**: Gin
- **ORM**: GORM
- **データベース**: PostgreSQL 15
- **AI**: Google Gemini API (Vision + Text)
- **認証**: JWT (golang-jwt/jwt)
- **アーキテクチャ**: Clean Architecture

### フロントエンド
- **言語**: TypeScript
- **フレームワーク**: React 18
- **ビルドツール**: Vite
- **状態管理**: React Context + Hooks
- **スタイリング**: Tailwind CSS
- **3D**: Three.js
- **国際化**: カスタムi18nシステム

### インフラ・DevOps
- **CI/CD**: GitHub Actions
- **コンテナ**: Docker
- **デプロイ**: 
  - バックエンド: Google Cloud Run
  - フロントエンド: Vercel
  - データベース: Cloud SQL (PostgreSQL)
- **Build**: Cloud Build（cloudbuild.yaml）

### AI・機械学習
- **Gemini API**: マルチモーダルAI（画像分析、テキスト生成）
- **AI自律エージェント**: 
  - 出品エージェント（画像→商品情報自動生成）
  - 交渉エージェント（価格交渉の自動化）
  - 配送エージェント（配送準備の自動化）
